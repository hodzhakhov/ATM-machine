# ATM-machine

Система банкомата на C# с использованием базы данных PostgreSQL и гексагональной архитектуры

## Функционал

1. Пользователь
   * Создание счета
   * Промсмотр баланса счета
   * Снятие денег со счета
   * Пополнение счета
   * Просмотр истории операций

2. Администратор
   * Создание пользователей
